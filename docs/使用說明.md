# 使用說明

四個模組的簡易使用說明。所有資料只存在你的瀏覽器，右上角工具列可隨時「匯出／匯入／清除」。

共用觀念：

- 切換工具用上方導覽列，網址會變成 `#/fire`、`#/tranche`、`#/deviation`、`#/journal`、`#/settings`。
- 「設定」頁的參數（集中度、現金區間、豁免清單、分類法、分類上限）會被「分批建倉」與「組合偏離」共用。
- 「組合偏離」貼上的持股會存成共享資料，「分批建倉」可一鍵帶入組合總值。

---

## 模組 1：FIRE 試算（`#/fire`）

回答四個直覺數字：現在幾歲、手上有多少本金、每月能投入多少、退休後每年想花多少。工具用 1871 年以來的真實市場歷史（或蒙地卡羅模擬）算出：

- 大約幾歲能退休、樂觀（p90）與保守（p10）區間
- 退休後照這樣花的成功率
- 你的「自由數字」與離目標多遠
- 少花一點／多投一點／晚一點退的取捨

進階設定（可不碰）：退休前後分開配置股債、安全提領率、餘命、費用率、七種提領策略、退休期額外現金流。上方可切換「歷史回測」與「蒙地卡羅」。所有金額為今日購買力。

## 模組 2：分批建倉計算器（`#/tranche`）

把「想買多少」拆成幾批分批進場。

輸入：標的代號（選填）、現價、可用現金、目前已持有股數（選填）；目標配置可切換「直接填金額」或「佔組合 %」（後者用組合總值換算，可按「帶入持股」從共享持股資料自動帶入）；分批數 1–4，每批填比重 % 與觸發條件（自由文字）。

輸出：每批一張卡片，第一眼就是「買幾股、約多少錢」，卡片下緣是執行後狀態（累計持股、已投入、剩餘現金與現金占比燈號，對照設定頁現金舒適區間）。累計投入超過可用現金會標「現金不足」。完整的逐批數字表收在「詳細表格」折疊區。下方有資金分布堆疊條與摘要（總投入、總股數、平均成本）。

計畫可命名後儲存多個，隨時載入／刪除，並逐批勾選「已執行第 N 批」（勾選後該列高亮）。這是執行計算工具，不產出買賣建議。

## 模組 3：組合偏離視覺化（`#/deviation`）

逐欄輸入持股，看組合的實際長相與偏離。邊改邊算，不需要按任何「解析」按鈕。

**輸入方式**：左側是一張可編輯表格，每一列就是一檔。直接在欄位裡填：

| 欄位 | 範例 | 說明 |
|---|---|---|
| 代號 | `NVDA` | 自動轉大寫顯示 |
| 現值 | `1800000` | 該檔市值，純數字 |
| 目標% | `20` | 目標佔比，填百分比數字（20 = 20%，不是 0.2） |
| 分類層級 | `半導體 / 算力` | 點開有設定頁的分類清單可選，也能直接打字自訂 |
| 實際% | （自動） | 由現值÷組合總值即時算出，唯讀 |

底部按鈕：「＋ 新增持股」加一列、「＋ 新增現金列」自動帶入代號＝現金、「載入示範資料」、「清空」。表格最下方顯示組合總值與目標佔比合計（偏離 100% 會以紅色提示）。每一格改動都會即時更新右側診斷並存入共享持股資料。

**整批貼上**：不想逐格打字，展開「整批貼上（Excel / Sheets / CSV）」，從試算表直接複製整個範圍貼進來（欄位順序：代號、現值、目標%、分類層級；Tab 或逗號分隔，表頭自動跳過）。「併入現有」以代號比對、同代號覆蓋現值；「取代全部」整份換新。日常更新市值時，貼上→併入，一次完成。

現金：用「＋ 新增現金列」或自行把某列代號填 `現金`／`CASH`，會納入組合總值與現金燈號。

**輸出**：

- 分類層級 treemap（磚塊大小依現值、顏色依分類法）
- 每檔「實際 vs 目標」偏離橫條（偏離 > 5 個百分點高亮）
- 觸發點燈號：單一標的超過集中度觸發點（在豁免清單者顯示「豁免」標籤而非警示）、現金低於/高於舒適區間、單一分類超過上限

門檻全部讀設定頁參數。此模組只做診斷呈現，不產出任何買賣建議。

## 模組 4：交易日誌統計（`#/journal`）

記下每筆買賣，回看自己的決策品質。

**單筆輸入（主要方式）**：左側「新增交易」表單——日期預設今天、代號、買/賣切換、股數、價格；賣出時可從下拉選賣出類型（停利/停損/再平衡/換股/調整部位）；決策評分與三層判讀完整度用按鈕點選，不必打符號。按「新增這筆」即入帳，表單自動清空可連續輸入（在代號或價格欄按 Enter 也能送出）。

**批次匯入（進階折疊）**：欄位（8 欄）：日期, 代號, 買/賣, 股數, 價格, 賣出類型(可空), 決策評分 ✅/⚠️/❌ 或 好/普/差(可空), 三層判讀完整度 0–3(可空)。日期接受 `YYYY-MM-DD` 或 `YYYY/MM/DD`。按「解析並累加」把新資料併入既有日誌，完全相同的列自動去重（適合逐期累積）。

**交易明細**：右側最上方是目前期間的明細表，每筆可「編輯」（行內修改後儲存）或「刪除」（單筆刪，不必清空重來）。

**統計輸出（季/年切換，可選期間或全部）**：交易筆數與金額、買賣分布、賣出類型分布、決策評分分布、三層判讀完整度趨勢（各期平均 0–3）、單一標的 30 天內重複交易標記。

資料持續累加存在瀏覽器，可「匯出 CSV」搬到 Excel，或「清空日誌」重來（建議先匯出備份）。

## 設定頁（`#/settings`）

可配置並儲存：

- 集中度觸發點：一般（預設 15%）、強檢視（預設 20%）
- 現金舒適區間：下緣（預設 10%）、上緣（預設 25%）
- 單一分類上限（預設 30%）
- 豁免標的清單（輸入代號新增；超過集中度但在清單中的標的只顯示「豁免」標籤）
- 資產分類法：AI 產業鏈六層 或 自訂分類

按「儲存設定」生效，「回復預設值」還原。
